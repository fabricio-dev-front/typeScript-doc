# COMPILADOR

- Para deixar sempre compilando o ts e nao precisarmos ficar sempre rodando 'tsc', utilizamos: `tsc -w`

# CONFIG TS

- `esnext` dentro do target em tsconfig.json ser para o ts utlizar a versao mais atual do javascript.
- `strict` serve para deixar 'rigoroso' o ts, nao deixar passar erros.
