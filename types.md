## Métodos e propriedades mais frequêntes utilizados:

# String

| Método          | Descrição                                     |
| --------------- | --------------------------------------------- |
| `length`        | Retorna a quantidade de caracteres.           |
| `toUpperCase()` | Converte o texto para maiúsculas.             |
| `toLowerCase()` | Converte o texto para minúsculas.             |
| `trim()`        | Remove espaços no início e no fim.            |
| `includes()`    | Verifica se um texto contém outro.            |
| `startsWith()`  | Verifica se começa com um determinado texto.  |
| `endsWith()`    | Verifica se termina com um determinado texto. |
| `replace()`     | Substitui parte do texto.                     |
| `split()`       | Divide a string em um array.                  |
| `slice()`       | Extrai uma parte da string.                   |
| `substring()`   | Retorna uma parte da string entre índices.    |
| `indexOf()`     | Retorna a posição de um texto.                |

# Number

| Método          | Descrição                              |
| --------------- | -------------------------------------- |
| `toString()`    | Converte o número para string.         |
| `toFixed()`     | Define a quantidade de casas decimais. |
| `parseInt()`    | Converte uma string em número inteiro. |
| `parseFloat()`  | Converte uma string em número decimal. |
| `Math.round()`  | Arredonda para o inteiro mais próximo. |
| `Math.floor()`  | Arredonda para baixo.                  |
| `Math.ceil()`   | Arredonda para cima.                   |
| `Math.random()` | Gera um número aleatório.              |
| `Math.max()`    | Retorna o maior número.                |
| `Math.min()`    | Retorna o menor número.                |

# Array

| Método        | Descrição                                              |
| ------------- | ------------------------------------------------------ |
| `length`      | Retorna a quantidade de elementos.                     |
| `push()`      | Adiciona um elemento no final.                         |
| `pop()`       | Remove o último elemento.                              |
| `shift()`     | Remove o primeiro elemento.                            |
| `unshift()`   | Adiciona um elemento no início.                        |
| `splice()`    | Adiciona, remove ou substitui elementos.               |
| `slice()`     | Cria uma cópia de parte do array.                      |
| `includes()`  | Verifica se um elemento existe.                        |
| `indexOf()`   | Retorna a posição de um elemento.                      |
| `find()`      | Retorna o primeiro elemento que atende a uma condição. |
| `findIndex()` | Retorna o índice do primeiro elemento encontrado.      |
| `filter()`    | Retorna um novo array com os elementos filtrados.      |
| `map()`       | Cria um novo array transformando os elementos.         |
| `forEach()`   | Percorre todos os elementos.                           |
| `reduce()`    | Reduz o array a um único valor.                        |
| `some()`      | Verifica se pelo menos um elemento atende à condição.  |
| `every()`     | Verifica se todos os elementos atendem à condição.     |
| `sort()`      | Ordena os elementos.                                   |
| `reverse()`   | Inverte a ordem dos elementos.                         |
| `join()`      | Junta os elementos em uma string.                      |

# Object

| Método             | Descrição                                       |
| ------------------ | ----------------------------------------------- |
| `Object.keys()`    | Retorna um array com as chaves do objeto.       |
| `Object.values()`  | Retorna um array com os valores do objeto.      |
| `Object.entries()` | Retorna um array de pares `[chave, valor]`.     |
| `Object.assign()`  | Copia ou combina objetos.                       |
| `Object.hasOwn()`  | Verifica se uma propriedade pertence ao objeto. |
| `Object.freeze()`  | Impede alterações no objeto.                    |

# Document (DOM)

| Método               | Descrição                                                 |
| -------------------- | --------------------------------------------------------- |
| `getElementById()`   | Busca um elemento pelo `id`.                              |
| `querySelector()`    | Busca o primeiro elemento que corresponde ao seletor CSS. |
| `querySelectorAll()` | Busca todos os elementos que correspondem ao seletor CSS. |
| `createElement()`    | Cria um novo elemento HTML.                               |
| `appendChild()`      | Adiciona um elemento como filho.                          |
| `append()`           | Adiciona um ou mais elementos ao final.                   |
| `remove()`           | Remove um elemento do DOM.                                |
| `setAttribute()`     | Define um atributo no elemento.                           |
| `getAttribute()`     | Obtém o valor de um atributo.                             |
| `addEventListener()` | Adiciona um evento ao elemento.                           |
| `textContent`        | Obtém ou altera apenas o texto do elemento.               |
| `innerHTML`          | Obtém ou altera o HTML interno do elemento.               |
| `classList.add()`    | Adiciona uma classe CSS.                                  |
| `classList.remove()` | Remove uma classe CSS.                                    |
| `classList.toggle()` | Adiciona ou remove uma classe automaticamente.            |
| `style`              | Altera estilos CSS diretamente pelo JavaScript.           |
